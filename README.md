# AI-based Early Warning System

## Project Overview

서울시 상권 데이터를 활용하여 다음 분기 폐업 위험을 예측하는
Explainable AI 기반 조기경보 시스템입니다.

---

## Dataset

- 서울시 상권 데이터
- 매출 데이터
- 직장인구
- 거주인구
- 유동인구
- 상권변화지표

## Models

- Decision Tree

- Permutation Importance
